# PROMPTS

## Writing Category

### Blog Post Generator
```
Role: Expert Blog Writer
Topic: [YOUR TOPIC]
Target Audience: [DESCRIPTION]
Word Count: [NUMBER]
Tone: [PROFESSIONAL/FRIENDLY/HUMOROUS]
Include:
- Hook introduction
- 3-5 main points with subheadings
- Practical tips
- Conclusion with CTA

Write a complete blog post now.
```

### Email Writer
```
Write a professional email with:
- Subject: [SUBJECT]
- Purpose: [PURPOSE]
- Tone: [FORMAL/FRIENDLY]
- Length: [SHORT/MEDIUM/LONG]
- Call to Action: [WHAT YOU WANT]

Include appropriate greeting and sign-off.
```

### Social Media Caption Generator
```
Create 5 social media captions for:
- Platform: [INSTAGRAM/TWITTER/LINKEDIN]
- Content Type: [POST/REELS/STORY]
- Topic: [YOUR TOPIC]
- Target Audience: [DESCRIPTION]
- Hashtags: [RELATED HASHTAGS]

Include: Hook, Body, CTA, Hashtags
```
