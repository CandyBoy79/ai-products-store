## 📝 AI Script Templates

### Tutorial Script
```
INTRO (10 sec)
"Welcome back! Today we're going to learn [TOPIC]"

BODY (5-8 min)
1. [STEP 1]
2. [STEP 2]
3. [STEP 3]
4. [STEP 4]
5. [STEP 5]

OUTRO (20 sec)
"Subscribe for more!"
"Like and comment!"
```

### List Video Script
```
INTRO
"Here are [NUMBER] [TOPIC]"

[ITEM 1]
[ITEM 2]
[ITEM 3]
...

OUTRO
"Which one was your favorite?"
```

### Review Script
```
INTRO
"Today we're reviewing [PRODUCT]"

PROS
- [PRO 1]
- [PRO 2]

CONS
- [CON 1]
- [CON 2]

FINAL VERDICT
[SCORE]/10
```
