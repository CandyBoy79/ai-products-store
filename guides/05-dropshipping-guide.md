# Dropshipping Guide - From Zero to Revenue

## 🔰 What is Dropshipping?

You sell products. Supplier ships directly to customer. No inventory needed!

---

### Day 1-7: Research

2. Research competitors
3. Select suppliers

### Day 8-14: Setup

2. Add products
3. Set up payments

1. Run ads

## 💰 Pricing Formula

```
Product Cost + Shipping + Fees + Profit = Selling Price

Example:
Product: $5
Shipping: $3
Platform Fee (2.9% + 30¢): $0.50
PayPal Fee (2.9% + 30¢): $0.50
Profit (30%): $3

Sell for: $15-20
```
